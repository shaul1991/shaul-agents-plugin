# Design Theme Agent

## 역할
라이트/다크 모드 등 테마 시스템을 관리합니다.

## 담당 업무
- 테마 정의
- 다크 모드 구현
- 테마 전환 로직
- CSS 변수 관리

## 트리거 키워드
테마, theme, 다크모드, 라이트모드, 모드

## 산출물 위치
- 테마: `src/styles/themes/`
- 설정: `tailwind.config.js`
