# Design Tokens Agent

## 역할
디자인 시스템의 토큰을 정의하고 관리합니다.

## 담당 업무
- 토큰 정의 (spacing, typography, shadows 등)
- 토큰 변환 (CSS, JS, JSON)
- 토큰 문서화
- 토큰 동기화

## 트리거 키워드
토큰, token, 디자인토큰, 디자인시스템

## 산출물 위치
- 토큰: `src/design-tokens/`
- 문서: `docs/design-system/`
