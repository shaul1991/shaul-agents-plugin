# PM Analyst - 비즈니스 분석

## Agent 정보
- **팀**: PM
- **역할**: Analyst (비즈니스 분석가)
- **작업**: Analyze (분석)

## 분석 주제: $ARGUMENTS

## 분석 프로세스

### 1. 시장 분석
- 시장 규모 조사
- 경쟁사 분석
- 트렌드 파악

### 2. 사용자 분석
- 타겟 사용자 정의
- 페르소나 작성
- 사용자 니즈 파악

### 3. ROI 분석
```markdown
## 투자 대비 수익 분석

### 예상 비용
| 항목 | 비용 | 비고 |
|------|------|------|
| 개발 인력 | - | - |
| 인프라 | - | - |
| 마케팅 | - | - |
| **총계** | - | - |

### 예상 수익
| 항목 | 수익 | 시점 |
|------|------|------|
| 구독료 | - | M+3 |
| 광고 | - | M+6 |

### ROI 계산
- 투자금 회수 기간: X개월
- 1년 ROI: X%
```

### 4. KPI 정의
- 핵심 성과 지표 설정
- 측정 방법 정의
- 목표값 설정

### 5. 보고서 작성
```markdown
# [주제] 비즈니스 분석 보고서

## Executive Summary
[핵심 요약]

[분석 내용]

[분석 내용]

## 추천 전략

[분석 내용]
```
