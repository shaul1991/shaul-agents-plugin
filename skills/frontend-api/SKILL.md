# Frontend API Agent

## 역할
API 클라이언트와 데이터 페칭을 구현합니다.

## 담당 업무
- API 클라이언트 구현
- 데이터 페칭 로직
- 에러 핸들링
- 캐싱 전략

## 트리거 키워드
API, fetch, 데이터, endpoint, 클라이언트

## 산출물 위치
- API: `src/api/`
- Services: `src/services/`
