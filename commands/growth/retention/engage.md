# Retention Manager - 참여도 향상

> **Shaul Corporation 원칙 적용**
>
> 사용자 유지와 참여도 향상을 통해 지속 가능한 성장을 추구합니다.

사용자 유지율을 분석하고 참여도를 향상시킵니다.

## 실행 단계

### 1단계: 리텐션 현황 분석

다음 정보를 수집하세요:

1. **제품/서비스**: 분석 대상
2. **현재 리텐션**: D1, D7, D30 리텐션
3. **핵심 행동**: 유지에 영향을 주는 행동

인자로 제공된 경우: $ARGUMENTS

### 2단계: 코호트 분석

```markdown
## 코호트 리텐션 분석

### 주간 코호트 리텐션
| 코호트 | Week 0 | Week 1 | Week 2 | Week 4 | Week 8 |
|--------|--------|--------|--------|--------|--------|
| W1 | 100% | - | - | - | - |
| W2 | 100% | - | - | - | - |

### 리텐션 커브 분석
- 급격한 이탈 구간: [시점]
- 안정화 시점: [시점]
- 목표 리텐션: [수치]
```

### 3단계: 이탈 원인 분석

```markdown
## 이탈 분석

### 이탈 시점별 원인
| 시점 | 주요 원인 | 비율 | 개선 방안 |
|------|----------|------|----------|
| D1 | 온보딩 미완료 | - | - |
| D7 | 핵심 가치 미경험 | - | - |
| D30 | 습관 미형성 | - | - |

### 이탈 사용자 특성
- 행동 패턴
- 세그먼트 특성
```

### 4단계: 참여도 향상 전략

```markdown
## 참여도 향상 전략

### 1. 온보딩 최적화
- Aha Moment 단축
- 핵심 기능 가이드
- 진행률 시각화

### 2. 습관 형성
- 트리거 설계
- 가변 보상
- 투자 유도

### 3. 재참여 캠페인
- 푸시 알림 전략
- 이메일 시퀀스
- 인앱 메시지

### 4. 커뮤니티/소셜
- 사용자 간 연결
- 소셜 프루프
- 경쟁/협력 요소
```
