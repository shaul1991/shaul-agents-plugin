# User Acquisition - 채널 최적화

> **Shaul Corporation 원칙 적용**
>
> 데이터 기반 최선의 판단으로 채널을 최적화합니다.

사용자 획득 채널을 분석하고 최적화합니다.

## 실행 단계

### 1단계: 채널 현황 파악

다음 정보를 수집하세요:

1. **현재 채널**: 운영 중인 획득 채널
2. **채널별 지표**: 유입, 전환, CAC
3. **예산**: 마케팅 예산

인자로 제공된 경우: $ARGUMENTS

### 2단계: 채널 분석

```markdown
## 채널 성과 분석

| 채널 | 유입 | 전환율 | CAC | LTV | LTV/CAC |
|------|------|--------|-----|-----|---------|
| Organic Search | - | - | - | - | - |
| Paid Search | - | - | - | - | - |
| Social Organic | - | - | - | - | - |
| Social Paid | - | - | - | - | - |
| Referral | - | - | - | - | - |
| Email | - | - | - | - | - |
| Content | - | - | - | - | - |
```

### 3단계: 채널 최적화 제안

```markdown
## 최적화 제안

### 고성과 채널 (LTV/CAC > 3)
- 예산 증대 권장
- 스케일업 전략

### 저성과 채널 (LTV/CAC < 1)
- 최적화 또는 중단 검토
- A/B 테스트 제안

### 미개척 채널
- 테스트 권장 채널
- 예상 ROI
```

### 4단계: 채널 믹스 제안

```markdown
## 권장 채널 믹스

| 채널 | 현재 비중 | 권장 비중 | 예상 ROI |
|------|----------|----------|---------|
| - | - | - | - |
```

### 5단계: 산출물 생성

`docs/growth/acquisition/channel-analysis.md` 생성

## 주의사항

- LTV/CAC 비율 3 이상 유지 목표
- 채널 다각화로 리스크 분산
- 단기/장기 채널 밸런스 유지
