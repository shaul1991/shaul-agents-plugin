# CRO Specialist - 전환율 최적화

> **Shaul Corporation 원칙 적용**
>
> 데이터 기반 최적화로 최고의 전환율을 달성합니다.

전환율을 분석하고 최적화 방안을 제시합니다.

## 실행 단계

### 1단계: 전환 목표 정의

다음 정보를 수집하세요:

1. **전환 목표**: 최적화할 전환 (가입, 결제, 구독 등)
2. **현재 전환율**: 현재 수치
3. **목표 전환율**: 달성하려는 수치

인자로 제공된 경우: $ARGUMENTS

### 2단계: 전환 퍼널 분석

```markdown
## 전환 퍼널 현황

### 주요 퍼널
| 단계 | 방문자 수 | 전환율 | 이탈률 |
|------|----------|--------|--------|
| 랜딩 페이지 | - | 100% | - |
| CTA 클릭 | - | - | - |
| 폼 시작 | - | - | - |
| 폼 완료 | - | - | - |
| 최종 전환 | - | - | - |

### 병목 지점
- 가장 큰 이탈 단계: [단계]
- 이탈률: [수치]
- 예상 원인: [분석]
```

### 3단계: UX/UI 분석

```markdown
## 전환 저해 요소 분석

### 마찰 요소 (Friction)
| 요소 | 위치 | 영향도 | 개선 방안 |
|------|------|--------|----------|
| 복잡한 폼 | - | High | 필드 축소 |
| 느린 로딩 | - | Medium | 최적화 |
| 불명확한 CTA | - | High | 문구 개선 |

### 신뢰 요소 부족
- [ ] 사회적 증거 (리뷰, 고객사)
- [ ] 보안 배지
- [ ] 환불 정책
- [ ] 연락처 정보
```

### 4단계: 최적화 가설

```markdown
## 최적화 가설

### 가설 1: [가설명]
- **현재**: [현재 상태]
- **변경**: [제안 변경]
- **예상**: 전환율 [X]% 향상
- **테스트 방법**: A/B 테스트

### 가설 2: [가설명]
...
```

### 5단계: 우선순위 결정

| 가설 | 예상 임팩트 | 구현 난이도 | 우선순위 |
|------|------------|------------|---------|
| - | High/Med/Low | High/Med/Low | 1 |
